Иконки PWA — как обновить
=========================

ПОЧЕМУ ВИДНА СТАРАЯ КРУГЛАЯ ИКОНКА?
------------------------------------
1. Service Worker раньше кэшировал PNG «навсегда» — исправлено в sw.js v3.
2. purpose "maskable" в manifest обрезал иконку в КРУГ — убрано.
3. Если приложение УЖЕ установлено на рабочий стол — иконка
   сохраняется в системе. Нужно УДАЛИТЬ приложение и установить заново.

КАК ПОМЕНЯТЬ ИКОНКУ
-------------------
1. Замените файл:  public/icons/icon-source.png  (ваш логотип Alvora)
2. Выполните:      python3 scripts/resize_pwa_icons.py
3. В Chrome: F12 → Application → Service Workers → Unregister
              → Storage → Clear site data
4. Жёсткое обновление: Cmd+Shift+R
5. Если PWA уже установлено: удалите ярлык с рабочего стола,
   затем установите снова через Chrome.

Проверка: откройте в браузере
  /icons/icon-512.png?v=3
должен быть ваш логотип Alvora, а не розовый круг.
