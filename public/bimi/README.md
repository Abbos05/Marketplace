# BIMI-логотип для Gmail

Положите сюда файл `logo.svg` (SVG Tiny 1.2, квадрат с отступами).

Источник: `public/icons/icon-source.png` — конвертируйте в SVG (Figma, Illustrator, https://convertio.co/png-svg).

DNS (пример):

```
default._bimi.alvoraplace.ru  TXT  "v=BIMI1; l=https://alvoraplace.ru/bimi/logo.svg;"
```

Подробнее: `docs/email-sender-avatar.md`
