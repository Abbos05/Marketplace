# ALVORA Marketplace
ALVORA Marketplace - платформа для онлайн-торговли между продавцами и покупателями. Основные модули: каталог, корзина, заказы, оплата, личные кабинеты...

## Описание

ALVORA — это современная платформа для онлайн-торговли, объединяющая продавцов и покупателей в удобной экосистеме. Проект направлен на создание безопасного и интуитивно понятного маркетплейса с широкими возможностями для бизнеса.

## Основные возможности
- Регистрация и управление профилями (покупатель/продавец)
- Каталог товаров с фильтрацией и поиском
- Корзина и оформление заказов
- Система оплаты и отзывов
- Личный кабинет продавца (добавление/редактирование товаров, статистика)

## Технологии
- Backend: PHP(Laravel)
- Frontend: React
- База данных: MySql
- Еще будут изменение

# 🛍️ Маркетплейс (ALVORA)

> Веб-приложение для организации торговой площадки с полным циклом покупки и продажи товаров.

## 📋 О проекте

Маркетплейс предоставляет возможность покупателям просматривать каталог товаров, добавлять их в корзину, оформлять заказы с онлайн-оплатой, оставлять отзывы. Продавцы могут управлять своими товарами, просматривать статистику продаж и формировать отчёты. Администратор и модератор контролируют товары и отзывы, настраивают комиссии и промокоды.

## 🗄️ Структура базы данных (19 таблиц, 3NF)

| Таблица | Описание |
|---------|----------|
| `users` | Пользователи (роли, аутентификация, профиль) |
| `seller_profiles` | Профили продавцов (магазин, ИНН, адрес самовывоза) |
| `regions` | Регионы с временем доставки |
| `categories` | Иерархические категории товаров |
| `products` | Товары (статус модерации, комментарий модератора) |
| `product_variants` | Варианты товаров (SKU, цена, скидка, остаток) |
| `product_images` | Изображения товаров/вариантов |
| `favorites` | Избранное пользователя |
| `cart_items` | Корзина (user_id, variant_id, quantity) |
| `orders` | Заказы (номер, статус, сумма, адрес, оплата) |
| `payments` | Платежи (ответы от платёжных шлюзов) |
| `order_items` | Позиции заказа (цена на момент покупки, комиссия) |
| `reviews` | Отзывы на товары (с модерацией) |
| `seller_reviews` | Отзывы на продавцов |
| `conversations` | Диалоги (покупатель↔продавец) |
| `messages` | Сообщения в диалогах |
| `promocodes` | Промокоды (тип, значение, ограничения) |
| `promocode_usages` | Использование промокодов в заказах |
| `commission_rates` | Ставки комиссии по категориям |
| `transactions` | Финансовые транзакции |

## 📁 Установка и запуск (по документации)

### Требования
- PHP ≥ 8.2, Composer, Node.js ≥ 20, MySQL ≥ 8.0
- Git

### Шаги
```bash
# 1. Клонирование
git clone https://github.com/your/repository.git
cd marketplace

# 2. Установка PHP-зависимостей
composer install

# 3. Настройка окружения
cp .env.example .env
php artisan key:generate

# 4. Настройка БД в .env (DB_DATABASE, DB_USERNAME, DB_PASSWORD)
php artisan migrate --seed

# 5. Установка фронтенда
npm install
npm run build

# 6. Запуск сервера
php artisan serve
